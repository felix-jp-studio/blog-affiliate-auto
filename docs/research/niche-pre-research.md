# ニッチ事前リサーチ（2026-07-12）

ブログアフィリエイト全自動化・12 ヶ月収益化向けのニッチ 3 案比較。

## 調査条件

| 項目     | 値                                   |
| -------- | ------------------------------------ |
| 目標月収 | 最低 5 万円 / ベスト 10 万円         |
| 作業時間 | 平日 1 時間/日（週 5〜9 時間）       |
| 技術     | TypeScript / Python 可               |
| 記事作成 | 全自動（AI 生成 → 投稿、リスク承知） |
| 対象市場 | 日本国内                             |
| ジャンル | 未決定（本調査で 3 案に絞る）        |

## 結論

|     順位      | ニッチ                                       | 評価                                       |
| :-----------: | -------------------------------------------- | ------------------------------------------ |
| **1（推奨）** | 格安 SIM × 光回線 × お困り系                 | 実績データ・自動化・単価のバランスが最良   |
|       2       | IT リスキル（プログラミングスクール × 副業） | 高単価だが SEO 競合と E-E-A-T がやや厳しい |
|       3       | ビジネス SaaS・副業ツール比較                | AI 相性最高だが成約数・PV が必要           |

---

## 評価軸の定義

| 軸             | 説明                                               |
| -------------- | -------------------------------------------------- |
| 自動化適性     | 公開情報ベースで AI 生成 → 投稿まで回せるか        |
| 5 万到達難易度 | 12 ヶ月で月 5 万円（ASP 手数料後）の現実性         |
| 規約リスク     | YMYL・薬機法・景表法・ASP 却下リスク               |
| 競合           | ヘッドワードの激しさ（ロングテールの豊富さも考慮） |

### 収益逆算の共通前提

- ASP 手数料約 30% を差し引いた**純利益 5 万円**には、粗利 **約 7.1 万円**が必要
- 記事経由 CVR は 1〜3% 程度を想定（ジャンルにより変動）

---

## 案 1：格安 SIM × 光回線 × お困り系（推奨）

### 概要

楽天モバイル・LINEMO・ahamo 等の格安 SIM と、NURO 光・au ひかり等の光回線を扱う比較・乗り換え・トラブルシューティング特化メディア。

### 選定理由

- 料金表・キャンペーン・MNP 手順は公式の**事実情報**が中心 → AI 全自動と相性が良い
- TypeScript/Python で**料金・キャンペーン差分の自動取得**パイプラインを構築しやすい
- 比較記事（乗り換え検討層）とお困り系（開通・速度・解約）の 2 層でネタが枯れにくい
- YMYL ではなく規約リスクが比較的低い

### 市場実績（参考）

| 事例               | 記事数 | 月間 PV             | 月間収益                | 備考                            |
| ------------------ | ------ | ------------------- | ----------------------- | ------------------------------- |
| 格安 SIM 特化      | 193 本 | 3〜4 万             | 平均約 19 万円          | 乗り換え KW 中心                |
| 光回線・WiMAX 特化 | 540 本 | 5,800               | 3〜4 万（繁忙期 6 万+） | 実体験記事多め                  |
| 光回線お困り系     | 146 本 | GSC クリック 518/月 | 仮成約 2 件             | AI 生成含む、比較導線で伸びしろ |

参考: [ラッコ M&A 格安 SIM](https://rakkoma.com/project/detail/22213)、[光回線お困り系](https://rakkoma.com/project/detail/22036)

### 自動化適性: 9/10

```
[公式・キャンペーン DB] → [料金差分検知] → [記事生成] → [WP 投稿] → [GSC 監視]
```

## 推奨 1 位の理由（案 1）

1. 200 記事・月 3 万 PV で月 20 万の事例があり、月 5 万は保守シナリオでも射程内
