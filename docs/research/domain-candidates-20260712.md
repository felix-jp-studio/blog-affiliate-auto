## 選定基準（調査時）

| 軸               | 重み   | 説明                                                    |
| ---------------- | ------ | ------------------------------------------------------- |
| テーマ適合       | 高     | SIM・光回線が伝わるか（Phase 1〜2 をカバー）            |
| 信頼感           | 高     | ASP 審査・読者のクリック率に影響                        |
| 長さ・読みやすさ | 中     | 20 字以内、ハイフンは 2〜3 個まで                       |
| SEO              | 低〜中 | 完全一致ドメインの効果は限定的だが、CTR・ブランドに寄与 |
| 商標リスク       | 高     | キャリア名（楽天・LINEMO 等）をドメインに含めない       |
| 取得可否         | 必須   | whois で未登録を確認                                    |

**TLD 方針（改訂）**: `.jp` にこだわらない。**`.com` + 独自ドメイン** で ASP 審査に対応。信頼感はコンテンツと必須ページで補う。

### 見送り

- **キャリア名入り**（`rakuten-sim-guide.*` 等）— 商標・ASP 規約リスク
- **格安のみ**（`kakuyasu-sim-*`）— 光回線フェーズでブランドがずれる
- **`.xyz` / `.site` 単体** — 信頼感が ASP 審査で不利になりやすい
- **`*.vercel.app` のみ** — 独自ドメインなしは審査不利

## GCP Cloud Domains での取得手順（Day 1）

1. GCP コンソール → **Cloud Domains** → ドメイン登録
2. `sim-hikari-guide.com` を検索・登録（連絡先・プライバシー設定）
3. DNS プロバイダに **Cloud DNS** を選択（マネージドゾーン自動作成）
4. **Vercel** にカスタムドメインを追加し、表示される A / CNAME を Cloud DNS に設定
5. HTTPS 有効化を確認

参照: [Cloud Domains ドメインの登録](https://cloud.google.com/domains/docs/register-domain?hl=ja)
