### A8 での確認手順

1. https://pub.a8.net/ を開き、**手動でログイン**
2. **レポート** → **成果レポート**（またはプログラム別レポート）
3. 期間を **前週（月〜日）** に設定
4. 各プログラムの **クリック数・成果件数・報酬額** をメモ

### バリューコマースでの確認手順

1. https://aff.valuecommerce.ne.jp/ を開き、**手動でログイン**
2. **成果レポート** → サイト `sim-hikari-guide.com`（siteId: 3776193）
3. 期間を **前週（月〜日）** に設定
4. LINEMO 等の **クリック数・成果件数・報酬額** をメモ
